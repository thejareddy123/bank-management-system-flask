# Bank Management System (Flask + MySQL)

A simple web-based **Bank Management System** built using **Flask** and **MySQL**.  
This project focuses on backend fundamentals with a clean and simple user interface.

---

#Features

- User Registration & Login
- Session-based Authentication
- View & Edit Profile
- Deposit Money
- Withdraw Money (with insufficient balance check)
- Transaction History
- Dashboard with Current Balance
- Clean UI with background image
- Logout functionality

---

# Tech

- Backend: Python (Flask)
- Database: MySQL
- Frontend: HTML, CSS
- Tools: Git, GitHub
